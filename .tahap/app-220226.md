## Persiapan Project

# Buat project baru dengan template blank (JavaScript)
npx create-expo-app@latest fcm-push-demo --template blank

# Library utama untuk notifikasi (wrapper Expo untuk FCM/APNs)
npx expo install expo-notifications

# Untuk akses device info (diperlukan expo-notifications)
npx expo install expo-device

# Untuk menyimpan token di local storage (opsional, best practice)
npx expo install @react-native-async-storage/async-storage

# Untuk navigasi (diperlukan jika ada deep link dari notifikasi)
npx expo install expo-linking

# Diperlukan oleh expo-notifications untuk mengambil projectId (yang ada di dalam Constants.expoConfig.extra.eas.projectId) untuk mengambil push token secara andal melalui EAS dari konfigurasi app.json
npx expo install expo-constants
